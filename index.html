Rules


Start the Game: When the game begins, one of the four colors (e.g., red, green, blue, yellow) will light up.

Repeat the Sequence: Click the same color that lit up. If you're correct, the game will add another color to the sequence.

Increasing Challenge: With each round, a new color will be added to the existing sequence. You need to remember and repeat the entire sequence in the correct order.

One Mistake Ends the Game: If you click the wrong color at any point, the game ends, and you'll need to restart from the beginning.

Progression: The sequence keeps growing, testing your memory until you make a mistake or decide to stop.

Winning: There is no set "end" to the gameâ€”challenge yourself to see how many levels you can complete!


How to Play


1. The game will show a color sequence, starting with one color.

2. Remember and press the colors in the correct order.

3. After each correct sequence, a new color will be added.

4. The game continues until the sequence is incorrect.
